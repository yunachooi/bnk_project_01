계좌입금

  • 부산은행 보유 계좌로만 가능 (외화계좌 입금 통화: USD, JPY, EUR, CNY, GBP, CHF, CAD, AUD)
  • 환전수령 → 원화계좌 재환전 시: 현찰매입율(50% 우대) 적용
  • 현찰배달 반송 후 재환전 시: 각 통화별 이벤트 우대율 적용
  • 비로그인 고객 자동환매 시: 현찰매입율 + 50% 우대 적용
  • 외화계좌 입금 후 재환전 시:
    • 입금 후 10일 이내 → 현찰매입율
    • 입금 후 10일 초과 → 전신환매입율
